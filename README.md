 Fake News Origin Tracker

 About Project

* This project detects whether a news article is **Fake or Real**
* It also shows **how news spreads** and finds the **possible origin**
* Uses **AI (NLP)** and **Graph Theory**

 Features

*  Fake vs Real news detection
*  Shows confidence score
*  News spread graph visualization
*  Finds origin source
*  Checks website credibility

 Tech Used

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* ML: Scikit-learn (TF-IDF)
* Graph: Network-based analysis


 Key Points

* Built full-stack AI project
* Used ML + Graph concepts
* Created interactive dashboard
* Deployed as live website
